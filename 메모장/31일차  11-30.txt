 데이터를 교환하기 위한 경량의 데이터 형식



 1 == "1"
true   -- 다음 코드는 자료형이 달라도 어떻게든 변환을 하고 나면 값이 같아진다

false == "0"
true -- false가 0으로, "0"이0으로 변환된 뒤에 비교

""  == []
true -- 빈 문자열은 false, 비어있는 배열[] false로 변환된 뒤에 비교

0 == []
true -- 0은 false, 비어있는 배열 []는 false로 변환된 뒤에 비교

변수를 만들 때에는 변수 앞에 키워드?를 사용

for 문 3가지 
기본적인 for 문:
for (초기식; 조건식; 증감식) {
    // 반복 실행될 코드
}

for...in 문 (객체 순회용):
for (변수 in 객체) {
    // 객체의 속성을 순회하는 코드
}

for...of 문 (배열 순회용):
for (변수 of 배열) {
    // 배열의 요소를 순회하는 코드
}

in array = index
of array = element

선언적 함수
-->선언적 함수는 function 키워드를 사용하여 정의되며, 함수의 이름이 주어집니다.  
익명적 함수
-->익명 함수는 변수에 함수를 할당하는 방식으로 정의됩니다. 함수에 이름이 없고 변수를 통해 참조됩니다.
